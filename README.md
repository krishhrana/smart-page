# Smart Page V0

This project is a frontend prototype for an AI-powered research workspace. It is built with **Next.js**, **Tailwind CSS**, and **Zustand** for state management.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the app.

## Features

- Project dashboard listing all projects
- Member session pages with personal AI chats
- Shared Resource Hub
- Collaborative Smart Doc

This repository only contains a frontend mock. Backend and AI integrations are left for future work.
